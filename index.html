jquery.betacode-convert.js

What is it?

jquery.betacode-convert.js is jQuery plugin for converting Beta Code text into Unicode. Beta Code is a convenient, ASCII-only encoding for Classical Greek used by the Thesaurus Linguae Graecae and other projects. With this plugin you can use Beta Code directly in web pages.

jquery.betacode-convert.js has evolved itself from an original Perl version of an online converter I created sometime around 1998.

How to get it

  • Download Version 1.0. Contains full (26K) and minified (6.6K) versions.
  • Fork the project from GitHub.

How to use it

The plugin creates a betacode2utf8 object with one method, convert(), which you can apply to any selector.

jquery.betacode-convert.js preserves markup, so you can apply it to a parent element without breaking the formatting of its children (Sappho, fr. 9).

*AI)Q' E)/GO XRUSOSTE/FAN' *)AFRO/DITA,
TO/NDE TO\N PA/LON LACO/HN.

Important! This plugin depends on the content to be converted already being in utf8. If you set the encoding of your webpages to anything else, it will almost certainly not work!
